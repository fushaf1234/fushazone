# Fusha Zone responsive redesign

A standalone, mobile-first static website based on the current Fusha Zone Academy content.

## What is included
- `index.html` — Complete responsive single-page site (Arabic RTL by default, with English language switch).
- Mobile navigation, smooth scrolling, Arabic/Qur'an course tabs, booking form that opens an email to `fushazone@gmail.com`, and private local comment previews.

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` to the repository root.
3. In **Settings → Pages**, choose **Deploy from a branch**, then select `main` and `/ (root)`.
4. GitHub will publish the site at your Pages URL.

No build tools or dependencies are required.
